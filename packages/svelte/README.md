# @jsonforms/svelte

Svelte 5 framework binding for [JSON Forms](https://jsonforms.io/).

Connects `@jsonforms/core` to Svelte 5's runes reactivity system. This package provides only the binding layer — a renderer set is required to actually render forms (see `@jsonforms/svelte-shadcn`).

## Installation

```bash
npm install @jsonforms/core @jsonforms/svelte svelte
```
